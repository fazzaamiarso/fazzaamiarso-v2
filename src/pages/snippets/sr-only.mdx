A utility class inspired from [tailwind css](https://tailwindcss.com/docs/screen-readers) `sr-only`. Mainly used for accessibility related.

Sometimes there's a need to hide an element visually, but still accessible to screen readers.

<MetaCodeBlock filename="sr-only utility class">
```css
/* Can be any name. I just prefer sr-only */
.sr-only {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  white-space: nowrap;
  border-width: 0;
}
```
</MetaCodeBlock>
